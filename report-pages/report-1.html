Eksperimen: Data Encryption

Penulis: Toki Monki • Tanggal: 12 Oktober 2025 • Level: Intermediate

Ringkasan

Eksperimen ini bertujuan memahami dasar-dasar enkripsi data menggunakan algoritma simetris dan asimetris. Lingkungan percobaan menggunakan Linux dengan Python dan OpenSSL.

Tujuan

  • Menerapkan enkripsi AES untuk data lokal.
  • Menggunakan RSA untuk enkripsi kunci.
  • Menguji dekripsi untuk verifikasi integritas data.

Langkah Percobaan

# Generate kunci RSA
openssl genrsa -out private.pem 2048
openssl rsa -in private.pem -outform PEM -pubout -out public.pem

# Enkripsi file
openssl rsautl -encrypt -inkey public.pem -pubin -in data.txt -out data.enc

# Dekripsi file
openssl rsautl -decrypt -inkey private.pem -in data.enc -out data_decrypted.txt

Hasil dan Analisis

File berhasil dienkripsi dan didekripsi dengan benar. Proses enkripsi RSA membutuhkan waktu lebih lama dibanding AES, namun memberikan keamanan tambahan melalui pertukaran kunci yang aman.

Kesimpulan

Enkripsi hybrid (AES + RSA) memberikan keseimbangan antara performa dan keamanan. Teknik ini dapat diterapkan untuk pengamanan komunikasi data sensitif pada aplikasi cybersecurity modern.
